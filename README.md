IronLinks is a simple and user-friendly website designed for a small family-owned ironworks business.
It provides an online portfolio of iron gates, grills, and railings, along with a built-in consultation booking system.

Features

Product catalog with categorized items (Gates, Grills, Railings)
3D demo models for enhanced product visualization
Online consultation booking form
Admin panel to view and manage bookings
Responsive and mobile-friendly design

Technology Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js
Database: Supabase
Deployment: Vercel
